Sails JS

Sails.js (o Sails) es un marco de aplicación web (MVC) desarrollado sobre el entorno Node.js, lanzado como software libre y de código abierto bajo la licencia MIT. Está diseñado para facilitar la creación de API y aplicaciones web personalizadas de grado empresarial. Al emular la arquitectura MVC de otros marcos, como Ruby on Rails, ofrece un patrón y familiaridad similares, lo que reduce la carga cognitiva al cambiar entre otros marcos. Este framework es utilizado como Back-end de la aplicacion Wintimeapp.com y Justinapp.com
